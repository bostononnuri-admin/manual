1장시스템 개요

예배 출석 보고 시스템은 보스턴 온누리교회의 여러 형태의 예배에 대한 주간 출석 인원을 담당자가 직접 제출하고, 행정간사가 이를 취합하여 예배인원을 관리하고, 운영위원회 보고서와 월간보고서를 생성을 돕는 온라인 도구입니다.

1-1. 무엇을 할 수 있나

① 예배 출석 제출 — 누구나

각 예배 담당자가 매 예배 출석 인원을 간단한 양식으로 제출합니다. 비밀번호 없이 바로 제출할 수 있습니다.

② 제출 내역 조회 달력 — 누구나

월별 달력으로 어떤 예배가 제출되었는지 한눈에 볼 수 있습니다. 수정 권한 없이 확인만 가능합니다.

③ 관리자 모드 — 관리자만

비밀번호로 로그인하면 제출된 내용을 수정·삭제할 수 있습니다. 로그인은 일정 시간 동안 유지됩니다.

1-2. 언제 쓰나
  • 매회의 모든 예배 후 — 각 부서 담당자가 출석 인원을 제출
  • 운영위원회 — 예배담당교역자가 운영위원회 보고서를 작성을 위해 참고
  • 매월 말 — 행정 간사가 월간보고서를 작성을 위해 참고
  • 상시 — 출석 기록 누락·오타 발견 시 예배담당교역자 및 행정간사가 수정·삭제
1-3. 대상별 역할 구분
역할할 수 있는 일
예배 담당자 (일반 사용자) 출석 인원 제출, 제출 내역 달력 조회
관리자 (로그인 후) 전체 출석 기록 조회·수정·삭제

3장일반 사용자 가이드

예배 담당자(간사·부서장 등)가 매주 출석 인원을 제출하는 방법을 설명합니다. 비밀번호 없이 누구나 바로 사용할 수 있습니다.

3-1. 출석 제출 방법
1

웹사이트 접속

휴대폰 또는 컴퓨터의 브라우저에서 아래 주소로 접속합니다. 로그인 없이 바로 제출 화면이 나타납니다.

http://ministry.bostononnuri.org - 사역 지원 시스템

http://attendance.bostononnuri.org - 예배 출석 보고 시스템

2

다섯 개 항목 입력

화면에 보이는 항목을 순서대로 채웁니다. 보고 날짜는 오늘 날짜가 자동으로 채워져 있어 건드릴 필요가 없습니다.

  • 예배 날짜 — 실제 예배가 드려진 날짜 (오늘이 아닐 수 있음)
  • 예배 종류 — 아래 목록에서 하나 선택
  • 인원수 — 예배별로 항목 구성이 다름 (3-2절 참고)
  • 서명 — 제출하는 본인 이름
3

제출하기 버튼 클릭

모든 항목이 채워졌는지 확인한 뒤 화면 하단의 제출하기 버튼을 누르면 끝입니다. "제출 완료" 화면이 뜨면 성공적으로 저장된 것입니다.

3-2. 예배별 인원수 항목

예배마다 세분화된 인원 항목이 다릅니다. 해당 예배를 선택하면 필요한 항목만 자동으로 표시됩니다.

예배 종류입력 항목
새벽예배 · 금요여성예배 현장 + 온라인 → 합계는 자동 계산
예꿈 · 꿈땅 · 중등부 · 고등부 학생 + 교역자·교사 + 그 외 → 합계는 자동 계산
그 외 모든 예배
(주일 1부, 주일 2부, 수요, 이음, 위윈, 위윈EM)		 총 인원 — 단일 숫자 하나만 입력
도움말 합계 칸은 직접 입력할 수 없으며, 각 항목에 숫자를 입력하면 자동으로 더해집니다. 숫자가 0이더라도 반드시 0을 입력해 주세요 (빈칸은 제출되지 않습니다).
3-3. 제출 내역 달력으로 확인하기

제출 화면 아래쪽에 제출 내역 확인 달력이 있습니다. 이 달력은 누구나 볼 수 있으며, 월별로 이동하면 어떤 날짜에 어떤 예배가 제출되었는지 태그 형태로 표시됩니다.

  • 왼쪽·오른쪽 화살표로 월을 이동합니다.
  • 날짜 칸의 네이비색 태그는 해당 날짜에 제출된 예배 종류를 뜻합니다.
  • 이 달력에서는 수정·삭제가 불가능하며 조회 전용입니다.
3-4. 이미 제출한 내용을 다시 제출할 때

같은 예배 날짜·같은 예배 종류로 이미 제출된 기록이 있으면 "중복 제출 확인" 창이 나타납니다. 이 때는 두 가지 항목을 추가로 입력해야 덮어쓸 수 있습니다.

①

본인 이름 (서명)

덮어쓰기를 요청하는 본인 이름을 입력합니다.

②

덮어쓰기 사유

왜 기존 기록을 바꾸는지 간단히 적습니다 (예: "숫자 오타 수정", "온라인 인원 빠졌었음").

주의 덮어쓰기를 확정하면 기존 숫자는 사라지고 새 숫자로 바뀝니다. 수정이력은 자동으로 저장됩니다.
4장관리자 가이드

관리자 모드에서는 모든 출석 기록을 조회하고, 잘못된 내용을 수정하거나 삭제할 수 있습니다.

4-1. 관리자 모드 들어가기
1

잠금 아이콘 누르기

화면 오른쪽 아래에 떠 있는 🔐 잠금 모양 동그란 버튼을 누릅니다. 이 버튼은 모든 화면에서 항상 보입니다.

2

비밀번호 입력

화면 가운데에 비밀번호 창이 뜹니다. 관리자 비밀번호를 입력하고 확인을 누릅니다. 취소하려면 취소 버튼을 누르면 됩니다.

3

관리자 뷰 진입

비밀번호가 맞으면 화면이 완전히 바뀌면서 관리자 뷰가 나타납니다. 일반 사용자용 제출 화면과 조회 달력은 이때 자동으로 숨겨집니다.

보안 안내 관리자 로그인은 일정 시간 동안 유지되며, 이후에는 비밀번호를 새로 입력해야 합니다. 브라우저를 새로고침하면 바로 로그인이 풀리니 주의해 주세요.
4-2. 관리자 뷰의 달력

관리자 뷰의 달력은 일반 달력과 비슷해 보이지만 각 태그를 누를 수 있습니다. 태그를 누르면 해당 제출 건의 상세 내용이 펼쳐지며, 그 자리에서 수정 또는 삭제 버튼이 나타납니다.

4-3. 출석 내용 수정하기
1

태그 선택 후 수정 버튼 클릭

달력에서 수정할 예배를 누르면 상세 영역이 펼쳐집니다. 그 안의 수정 버튼을 누르면 화면 아래쪽에서 슬라이드 창이 올라옵니다.

2

인원수 · 이름 · 사유 입력

새 인원수로 바꿔 입력한 뒤, 수정하는 본인 이름과 수정 사유를 적습니다. 수정 사유는 나중에 어떤 기록이 왜 바뀌었는지 추적하는 데 쓰이므로 구체적으로 적어 주세요.

3

저장 버튼 클릭

저장 버튼을 누르면 구글 스프레드시트의 해당 행이 즉시 갱신됩니다. 취소하려면 슬라이드 창의 취소 버튼을 누릅니다.

4-4. 출석 기록 삭제하기

수정과 동일한 방식으로 시작합니다. 달력에서 태그를 누르고 상세 영역의 삭제 버튼을 누르면 삭제 확인 슬라이드 창이 올라옵니다. 본인 이름과 삭제 사유를 반드시 적어야 하며, 최종 확인을 누르면 해당 행이 스프레드시트에서 완전히 제거됩니다.

주의 삭제는 되돌릴 수 없습니다. 기록 자체를 남기고 숫자만 바꾸고 싶다면 삭제 대신 수정을 쓰는 것이 좋습니다. 혹시 실수로 지웠다면 구글 스프레드시트의 버전 기록에서 복구가 가능하니 바로 행정 간사에게 알려 주세요.
4-5. 로그아웃

관리자 뷰 상단의 로그아웃 버튼을 누르면 즉시 로그인이 해제되고 일반 사용자 화면으로 돌아갑니다. 이 때 숨겨져 있던 제출 화면과 조회 달력이 다시 나타납니다.


5장   자주 묻는 질문

제출 화면에서 "보고 날짜"를 바꿀 수 있나요?
바꿀 수 없습니다. 보고 날짜는 제출하는 그 날짜로 자동 설정되며, 수정이 불가능한 항목입니다. 실제 예배가 드려진 날짜는 그 아래 예배 날짜 칸에서 직접 고르시면 됩니다.
제출했는데 다른 사람이 같은 예배를 또 제출하면 어떻게 되나요?
같은 예배 날짜·같은 예배 종류 기록이 있으면 "중복 제출 확인" 창이 먼저 뜹니다. 그 창에서 본인 이름과 덮어쓰기 사유를 입력해야만 기존 숫자가 새 숫자로 바뀝니다. 입력 없이 취소하면 기존 기록이 그대로 유지됩니다.
관리자 비밀번호를 잘못 입력하면 어떻게 되나요?
"비밀번호가 올바르지 않습니다" 안내가 뜨고, 다시 입력할 수 있습니다. 비밀번호 관리에 각별히 주의해 주시고, 외부에 노출되지 않도록 안전하게 보관해 주세요.
실수로 출석 기록을 삭제했어요. 복구할 수 있나요?
시스템 자체에는 복구 기능이 없지만, 모든 데이터가 저장되는 구글 스프레드시트의 버전 기록에서 이전 상태로 되돌릴 수 있습니다. 빠르게 행정 간사에게 알려 주세요. 스프레드시트의 파일 → 버전 기록 메뉴에서 삭제 직전 시점으로 되돌리면 복구됩니다.
일반 사용자도 달력에서 태그를 누르면 수정할 수 있나요?
아니요. 일반 사용자가 보는 달력은 조회 전용이라 태그를 눌러도 수정·삭제 메뉴가 나타나지 않습니다. 수정·삭제는 관리자 모드에 로그인한 상태에서만 가능합니다.
모바일 휴대폰으로도 쓸 수 있나요?
네. 휴대폰 브라우저에서도 모든 기능이 그대로 작동합니다. 화면 크기에 맞춰 자동으로 배치가 바뀝니다. 아이폰 홈 화면에 바로가기로 추가하면 앱처럼 빠르게 열 수 있습니다. (사파리에서 주소창 하단의 공유 → 홈 화면에 추가)
같은 예배를 두 사람이 동시에 제출하면 어떻게 되나요?
나중에 제출한 사람 화면에 "중복 제출 확인" 창이 뜹니다. 그 때 덮어쓰기 사유를 적고 확인하면 최신 제출 내용으로 바뀝니다. 두 사람 모두 단순 제출만 하고 중복 확인 창을 무시하면, 처음 제출된 기록이 그대로 남고 나중 시도는 반영되지 않습니다.
