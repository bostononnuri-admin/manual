1. 시스템 한눈에 보기

보스턴 온누리교회 행정재정 시스템은 교회의 지출·행사·결정 관련 업무를 한 곳에서 처리할 수 있는 웹 시스템입니다. 홈 화면에서 원하는 시스템을 선택합니다.

지출 및 결산신청서
영수증을 근거로 교회 지출에 대한 환급을 청구하거나 결산을 보고하는 시스템
승인 단계 최대 4단계 · $1,000 이상은 담임목사까지 승인
영수증 분실 보완
영수증이 분실되었거나, 없는 경우, 지출에 대한 카드 결제 내역을 팀장 승인을 통해, 영수증 대신 보완하는 서류 시스템. 현금으로 결제했으나 영수증이 없는 경우 재정간사에게 직접 문의해야 합니다.
승인 단계 1단계 (팀장) · 지출 및 결산신청서에 첨부 가능
행사계획서
사역팀의 예산 내 이미 계획된 행사·프로그램 중 전 성도를 대상으로 하는 행사를 사전에 기획하고 예산을 승인받는 시스템
승인 단계 3단계 · 담당교역자 → 행정목사 → 운영위 총무
결정의뢰서
사역팀이 예산 외에 추가 예산을 필요로 하는 사역을 진행하거나, 예산의 항목변경이 필요한 경우, 운영위원회의 공식 결정을 요청하는 시스템
승인 단계 3단계 · 담당교역자 → 행정목사 → 운영위 총무

2. 공통 사항 — 모든 시스템 공통

2-1. 4자리 비밀번호

지출 및 결산신청서, 영수증 분실 보완, 행사계획서, 결정의뢰서 모두 제출 직전에 4자리 숫자 비밀번호를 설정합니다. 이 비밀번호는 본인이 나중에 조회·수정할 때 사용됩니다. 반드시 메모해 두세요. 분실한 경우 본인이 직접 복구할 수 없으니, 행정 간사에게 문의하시면 별도의 안전한 절차로 도움을 받으실 수 있습니다.

2-2. 승인자 처리 결과

모든 시스템의 각 단계 승인자는 아래 세 가지 중 하나로 처리합니다.

처리의미이후 흐름
승인내용에 동의다음 단계 승인자에게 자동 전달 (마지막 단계면 최종 완료)
재고보완이 필요 (사유 작성 필수)신청자에게 사유 통보. 신청자가 수정 후 [재승인 요청]을 눌러야 다시 진행됨
반려진행 부적절 (사유 작성 필수)신청서 종료, 수정 불가. 필요 시 새로 작성

2-3. 이메일로 바로 처리 (승인자)

승인자에게는 이메일로 [승인] / [재고] / [반려] 버튼이 발송됩니다. 별도 로그인 없이 이메일에서 바로 처리 가능합니다.

2-4. 미결 알림 (승인자)

지출 및 결산 신청에 대한 승인이 지연될 경우, 최초 승인 요청일로부터 3일 및 7일 경과 시점에 미결 알림 이메일이 발송됩니다. 해당 이메일은 최초 발송된 승인 요청 이메일과 동일한 기능을 제공합니다.

2-5. 문서 복사·붙여넣기 주의

알려진 제한사항. 일부 문서에서 텍스트를 복사해 붙여넣을 때 띄어쓰기가 사라지거나 특수 공백 문자로 변환되는 경우가 있습니다. 붙여넣기 후 반드시 띄어쓰기를 직접 확인·수정해 주세요.

3. 지출 및 결산신청서

3-2. 특별 승인 케이스 3가지 (운영위원회 사전 승인 필요)

일반 제출 외에 다음 특별 케이스가 있습니다. 신청 시 해당 사항에 체크하면 자동 적용됩니다.

3-4. 신청자 — 재고·반려 대응

1재고를 받았을 때 — 수정 후 재승인 요청

홈 → [지출 및 결산신청서] → [확인 및 수정] → 사역팀 선택 → 본인 신청서 클릭 → 4자리 비밀번호 입력 → [수정하기]

내용 수정 후 저장한 다음, 반드시 [재승인 요청] 버튼을 눌러야 1단계부터 다시 시작됩니다.

첨부된 분실보완 서류 연동. 지출 및 결산신청서가 재고 처리되면 첨부된 영수증 분실 보완 서류도 자동으로 "재고" 상태가 되고, 재승인 요청 시 "제출완료"로 자동 복원됩니다.

2반려를 받았을 때

반려된 신청서는 더 이상 수정할 수 없습니다. 새 신청서를 작성해 주세요.

3-5. 승인자 가이드

1단계별 승인자 구성

  • 1단계 재정간사: 영수증·예산항목·재정코드 확인
  • 2단계 팀장: 사역팀 집행 내역 확인
  • 3단계 담당교역자: 사역 관점에서 승인
  • 4단계 담임목사: $1,000 이상일 때만 활성화

2이메일 처리

이메일의 [승인] · [재고] · [반려] 버튼 중 하나를 눌러 처리합니다. 재고·반려 시 별도 창에서 사유를 반드시 작성해야 합니다.

4. 영수증 분실 보완

영수증을 분실했거나 받지 못한 지출 건에 대해, 카드 결제 내역을 근거로 제출하는 보완 서류 시스템입니다. 승인되면 지출 및 결산신청서 작성 시 첨부 서류로 연결해 사용할 수 있습니다. 현금을 사용한 영수증 분실시에는 사용하실 수 없습니다. 이럴 경우 재정간사에게 문의해 주시기 바랍니다.

4-3. 지출 및 결산신청서와의 연동

1지출 및 결산신청서에서 분실보완 서류 첨부하기

지출 및 결산신청서 신청 화면의 [분실보완 서류 선택] 버튼으로 승인된 영수증 분실 보완 서류를 선택 첨부할 수 있습니다. 최종 승인 시 분실보완 서류는 지출 및 결산신청서 합본 문서의 마지막 페이지에 자동 병합됩니다.

연동 규칙
· 지출 및 결산신청서가 재고 처리되면 연결된 분실보완 서류도 "재고" 상태로 자동 변경
· 재승인 요청 시 분실보완 서류는 "제출완료"로 자동 복원
· 환급완료 시 연결된 분실보완 서류도 "환급완료"로 자동 처리

6. 결정의뢰서

사역팀이 예산 외 추가 예산을 필요로하는 사역을 계획하거나, 예산 항목 변경 필요시, 운영위원회의 공식 결정이 필요한 사안을 상정하기 위한 문서입니다. 결정의 과정과 근거를 공식 기록으로 남기는 목적이 있습니다.

사용이 적절한 경우. 새로운 사역·프로그램 신설, 기존 정책 변경, 사역팀 단독으로 결정하기 부담스러운 중요 사안, 여러 사역팀의 협력이 필요한 일.

6-3. 운영위 총무의 최종승인 — 결정 근거 작성

13단계 운영위 최종승인 = 결정 근거 작성 필수

3단계 운영위 총무가 [운영위 최종승인]을 누르면 별도 창이 열려 운영위원회 결정 근거를 반드시 입력해야 합니다. 이 내용은 운영위원회에서 어떤 논의를 거쳐 어떤 결정을 내렸는지를 서술합니다.

결정 근거는 최종 문서의 별도 항목으로 영구 보존되며, 신청자·담당교역자·재정팀장·재정간사 총 4명에게 문서가 첨부된 최종 알림이 발송됩니다.

버전생성 시점용도
1차 문서2단계 행정목사 승인 직후3단계 운영위 총무에게 회의 검토용으로 첨부
최종 문서3단계 운영위 최종승인 + 결정 근거 작성 완료 시4명에게 최종 통보용 (1차 문서는 자동 보관)

7. 관리자 기능 — 전 시스템 공통

행정목사, 행정간사는 별도의 관리자 권한으로 전 시스템의 모든 신청서를 조회·수정할 수 있습니다.

7-1. 관리자 패널 접근

1홈 → 우상단 관리자 버튼 → 관리자 비밀번호 입력

관리자 모드로 진입한 뒤, 4개 시스템 중 하나를 선택합니다. 각 시스템의 전체 신청서 목록을 볼 수 있습니다.

7-2. 관리자 권한 요약

권한설명
비밀번호 없이 조회모든 신청서를 비밀번호 없이 열람 가능
내용 수정최종 승인 이전: 자유 수정 / 지출 및 결산신청서의 환급완료 상태: 관리자만 수정 가능
환급완료 처리지출 및 결산신청서 최종 승인 후 재정간사가 환급완료로 마킹 (보라색 이메일 발송)
삭제관리자 이름과 사유 입력 필수. 별도 "삭제" 시트로 영구 보관 (복구 가능)

7-3. 자동 백업 스케줄

모든 시스템 데이터는 매일 새벽(미 동부시간)에 자동 백업됩니다. 실수로 삭제·변경되어도 백업본에서 확인할 수 있습니다.

시간시스템
새벽 3:00지출 및 결산신청서
새벽 3:15행사계획서
새벽 3:30영수증 분실 보완
새벽 3:45결정의뢰서

7-4. 수정 이력 추적

모든 시스템은 변경 이력이 자동 기록됩니다. 누가·언제·어떤 항목을 어떻게 바꿨는지 색상별로 구분되어 저장됩니다. (신규제출=파란, 수정=청록, 승인=초록, 재고=주황, 반려와 삭제=빨강, 환급완료=보라)

8. 자주 묻는 질문

제출했는데 이메일이 오지 않습니다.
스팸 메일함을 먼저 확인해 주세요. 그래도 없으면 행정 간사에게 신청번호와 함께 문의하시면 재발송해 드립니다.
영수증을 분실했는데 환급받을 수 있나요? (카드결제건만 해당)
네. 영수증 분실 보완 시스템으로 카드 결제 내역과 지출처 정보를 먼저 제출하고, 팀장 승인을 받으면 그 서류를 지출 및 결산신청서 작성 시 영수증 대신 첨부할 수 있습니다.
지출 금액이 $1,000 이상이면 무엇이 달라지나요?
지출 및 결산신청서의 4단계 담임목사 승인이 활성화됩니다. 1~3단계를 마친 뒤 담임목사 승인을 추가로 받아야 최종 완료됩니다.
반려된 신청서를 다시 살릴 수 있나요?
반려된 문서는 다시 진행할 수 없습니다. 내용을 보완해 새 신청서를 제출해 주세요. 기록은 시스템에 영구 보존됩니다.
재고 후 수정만 저장하면 자동으로 재승인이 되나요?
아닙니다. 수정을 저장한 뒤 반드시 [재승인 요청] 버튼을 눌러야 1단계부터 다시 시작됩니다. 지출 및 결산신청서, 행사계획서, 결정의뢰서 모두 동일합니다.
붙여넣은 텍스트의 띄어쓰기가 이상합니다.
일부 문서에서 복사한 텍스트는 띄어쓰기 정보가 손상된 상태로 복사됩니다. 붙여 넣은 후 화면에서 직접 확인·수정해 주세요.
최종 승인이 끝난 신청서를 나중에 다시 볼 수 있나요?
네. 신청자는 본인 비밀번호로, 관리자는 언제든지 조회 가능합니다. 최종 문서는 시스템별 폴더에 영구 보관됩니다.
첨부파일 크기 제한이 있나요?
개수 제한은 없으나 한 파일당 수십 메가바이트 이내를 권장합니다. 이미지 파일과 문서 파일 모두 가능합니다. 모두 원본 그대로 안전하게 보관됩니다.
