1-1.허브 페이지란 무엇입니까?

허브 페이지는 보스턴 온누리교회 성도라면 누구나 사용하는 사역 지원 시스템의 출발점입니다. 안내받은 주소로 한 번 접속하면, 4개의 운영 시스템(예배 출석 보고, 장소 예약, 행정·재정, 행사 신청)에 모두 들어갈 수 있습니다.

별도의 회원가입이나 로그인 없이 바로 사용할 수 있도록 만들어졌으며, 컴퓨터·휴대전화 등 어느 기기에서나 사용할 수 있습니다.

왜 허브 페이지가 필요한가요?
4개 시스템이 각각 다른 주소(URL)에 있어, 매번 주소를 외우거나 즐겨찾기를 따로 만들어야 하는 불편함 없이 허브 페이지 한 곳만 기억하면 모든 시스템으로 편리하게 이동할 수 있습니다.
1-2.어떻게 접속합니까?

웹브라우저(크롬·사파리·엣지 등)에서 아래 주소를 입력하면 됩니다. 주소는 외우기 쉬운 형태로 만들어져 있습니다.

📍 허브 페이지 주소
http://ministry.bostononnuri.org
1
브라우저 주소창에 주소 입력
크롬, 사파리, 엣지, 파이어폭스 등 어느 브라우저에서나 가능합니다. 주소창에 ministry.bostononnuri.org를 입력하고 엔터를 누르세요.
2
즐겨찾기에 추가하기 (권장)
한 번 방문하신 후에는 즐겨찾기(북마크)에 추가하면, 다음에는 클릭 한 번으로 바로 접속할 수 있습니다.
3
홈 화면에 추가 (휴대전화)
아이폰 사파리에서는 [공유] → [홈 화면에 추가] 메뉴로, 안드로이드 크롬에서는 [메뉴] → [홈 화면에 추가] 메뉴로 앱처럼 사용할 수 있습니다. 교회 로고가 아이콘으로 표시됩니다.
TIP 휴대전화에서 더 빠르게 사용하려면 [홈 화면에 추가]를 추천합니다. 앱처럼 한 번의 탭으로 열리며, 화면 전체를 사용해 더 편리하게 볼수 있습니다.
1-3.4개 시스템 한눈에 보기

허브 페이지에 접속하면 4개의 버튼이 보입니다. 각 버튼을 누르면, 해당 시스템으로 즉시 이동합니다.

01 · ATTENDANCE
예배 출석 보고 시스템
예배 담당자가 매주 출석 인원을 제출할 수 있습니다.
02 · RESERVATION
장소 예약
교회 내 공간(예배실, 회의실, 식당 등)을 온라인으로 예약할 수 있습니다. 20인 이상 행사는 담당 목회자의 승인이 필요합니다.
03 · ADMIN & FINANCE
행정·재정 시스템
지출결산, 영수증 분실 보완, 행사계획서, 결정의뢰서를 한 곳에서 작성하고 승인받습니다.
04 · SIGNUP
행사 신청 플랫폼
교회 행사의 참가자·봉사자 신청서를 자동으로 만들어, 신청 명단을 한 곳에 모아 관리합니다.

각 버튼을 한 번 누르면 해당 시스템으로 이동합니다. 브라우저의 [뒤로 가기] 버튼을 누르면, 언제든 허브 페이지로 돌아오실 수 있습니다.

1-4.일반적인 사용 과정

대부분의 경우 다음과 같은 과정으로 진행됩니다.

단계내용소요 시간
1허브 페이지 접속 (ministry.bostononnuri.org)약 5초
24개 시스템 버튼 중 사용 하고자 하는 버튼 누르기약 3초
3해당 시스템에서 작업 (제출·예약·신청 등)1~10분
4완료 후 브라우저 닫거나 [뒤로 가기]—
팁: 각 시스템의 자세한 사용 방법은 다음 챕터(Chapter 2~5)에서 안내드립니다. 본 매뉴얼북의 상단 [📖 Manual] 버튼을 누르면, 언제든 이 목차로 돌아오실 수 있습니다.
1-5.자주 묻는 질문
허브 페이지를 사용하지 않고 각 시스템에 바로 접속해도 되나요?
네, 가능합니다. 각 시스템은 독립적인 주소를 가지고 있어 직접 접속할 수 있습니다. 다만 4개 주소를 모두 외우거나 즐겨찾기에 따로 등록해야 하는 번거로움이 있어, 허브 페이지를 통해 편하게 접속할 수 있습니다 .

각 시스템 직접 주소:
· 예배 출석: attendance.bostononnuri.org
· 장소 예약: reservation.bostononnuri.org
· 행정·재정: claim.bostononnuri.org
· 행사 신청: signup.bostononnuri.org
로그인이 필요한가요?
허브 페이지 자체는 로그인이 필요하지 않습니다. 단, 각 시스템 내부에서는 일부 기능(예: 행정·재정의 승인자 처리, 장소 예약의 관리자 모드 등)에서 비밀번호가 필요할 수 있습니다. 자세한 내용은 각 챕터에서 안내드립니다.
스마트폰에서도 잘 작동하나요?
네, 모든 시스템이 스마트폰에 최적화되어 있습니다. 특히 허브 페이지는 아이폰 사파리 환경을 기준으로 설계되었으며, 안드로이드 크롬에서도 동일하게 작동합니다. [홈 화면에 추가] 기능을 활용하면 앱처럼 사용할 수 있습니다.
화면이 어둡게 보이거나 색상이 이상해 보입니다.
허브 페이지와 모든 가이드는 밝은 모드(라이트 모드)에 최적화되어 있습니다. 시스템 설정에서 다크 모드를 사용 중이더라도, 본 매뉴얼북과 모든 시스템은 자동으로 라이트 모드로 표시 되도록 설정되어 있습니다. 만약 색상이 이상해 보인다면 브라우저를 새로고침 해주세요.
매뉴얼북의 다음 챕터로 어떻게 이동하나요?
이 페이지 맨 아래의 [다음 챕터 →] 버튼을 누르거나, 위의 전체 목차에서 원하는 챕터를 직접 누르면 됩니다. 각 챕터 페이지의 상단에는 항상 [📖 Manual] 버튼이 있어 언제든 이 목차로 돌아올 수 있습니다.
